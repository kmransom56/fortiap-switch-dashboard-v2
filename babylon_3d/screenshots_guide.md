# Screenshots Guide for 3D Network Topology Lab

## Required Screenshots

### 1. Main 3D Visualization
- **URL**: http://localhost:3001
- **File**: `3d_main_interface.png`
- **Content**: Complete Babylon.js interface with sidebar and controls

### 2. Configuration Display
- **Command**: `python run_fortigate_discovery.py --config`
- **File**: `configuration_setup.png`
- **Content**: Show environment-based configuration

### 3. VSS Converter Help
- **Command**: `python vss_to_svg.py --help`
- **File**: `vss_converter_help.png`
- **Content**: Show available conversion options

### 4. Project Structure
- **File**: `project_structure.png`
- **Content**: Show organized directory structure

### 5. README Preview
- **File**: `readme_preview.png`
- **Content**: Show professional README documentation

## Screenshot Instructions

### Windows Users:
1. Use `Win + Shift + S` for Snipping Tool
2. Capture full application windows
3. Save as PNG for best quality
4. Name files as specified above

### Browser Screenshots:
1. Open http://localhost:3001
2. Use browser screenshot tools
3. Capture full page for complete interface
4. Include URL bar for context

## Adding to Repository

1. Create `screenshots/` directory in project root
2. Add screenshots with specified names
3. Update README.md to include screenshots
4. Commit and push to GitHub
